## Installation

1. Clone the repository:
```bash
git clone https://github.com/rohit-singh2005/Tasteat-table-booking-website.git
cd thetasteat
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
node server.js
```

4. Access the application:
Open your browser and navigate to `http://localhost:3000`
